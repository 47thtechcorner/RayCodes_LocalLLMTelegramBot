# RayCodes : LocalLLM Telegram Bot

Code for my Youtube Video : https://youtube.com/shorts/DehRH_eAlME?feature=share

[STEP 1]
Make a new .env file and paste your tokens like this: <br>
TELEGRAM_BOT_TOKEN=...<br>
TELEGRAM_CHAT_ID=....<br>

[STEP 2]
Add your stock symbols, local llm model downloaded and message frequency in config.json

[STEP 3]
Run the bot by running run_bot.bat file

[STEP 4]
To stop the bot press ctrl+C in running terminal and press Y to stop windows process
